Animación

Este ciclo está orientado para la realizaciónde películas de animación, elaboración de guiones, creación de escenarios, creación de fotogramas clave, realización de dibujos de intercalación, animación 2D y 3D y renderización.

Arquitectura Efímera

Este ciclo está orientado para el diseño de escenografías para teatro, cine etc..., diseño de espacios de ocio, diseño de espacios expositivos y dirección y coordinación de obras propias.

Cerámica Artística

Este ciclo está orientado para la realización de piezas cerámicas propias, organización y coordinación de grupos de trabajo y como trabajador dependiente de profesionales de nivel académico superior.

Cómic

Este ciclo está orientado para preparar especialistas en narración gráfica, la posibilidad de organizar y supervisar la producción y edición de cómics, elaboración de guiones, creación de obra original, realización de story-boards etc....

Escultura Aplicada al Espectáculo

Este ciclo está orientado para la realización de objetos tridimensionales para teatros, ópera, cine etc..., realización de estructuras y creación de objetos de atrezzo y utilería para cine, teatro, tv etc...

Fotografía

Este ciclo está orientado a el fotoperiodismo, cobertura gráfica para medios de comunicación, reportaje social, documentación de actos sociales y retratos, fotografía publicitaria, trabajo para agencias, fotografía de moda, encargos individuales etc...

Gráfica Interactiva

Este ciclo está orientado a la realización de productos interactivos, diseño y desarrollo web, creación de productos accesibles, colaboración especialista en equipos multidisciplinares y gestión de proyectos de equipo.

Gráfica Publicitaria

Este ciclo está orientado para la realización de productos gráficos impresos, realización de aplicaciones vinculadas a la comunicación gráfica, grafismo, etc...

Ilustración

Este ciclo está orientado para ser un especialista en ilustración y dibujante, organización y supervición de producciones, creación de ilustraciones originales para diferentes ámbitos, creación de proyectos de libros y desplegables etc...

Modelismo de Indumentaria

Este ciclo está orientado a la creación de alta-costura, prêt-à-porter, ropa deportiva, laboral, lencería y géneros de punto y piel.

Proyectos y Dirección de Obras de Decoración

Este ciclo está orientado a diseño de interiores, dirección de obras de interiorismo, asesoramiento en diseño de ventas de mobiliario, posibilidad de trabajar en estudios, empresas de mobiliario y organismos oficiales.

Técnicas Escultóricas

Este cilo está orientado a ser encargado de taller de producción artística de elementos ornamentales, realización de obra de elementos escultóricos destinados a funciones, proyección de obras y realización de obras de elementos escultóricos destinados a la recuperaciónd de patrimonio.
